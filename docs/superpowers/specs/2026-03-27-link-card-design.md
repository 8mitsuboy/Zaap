# LinkCard コンポーネント 設計ドキュメント

**日付**: 2026-03-27
**ステータス**: 承認済み

---

## 概要

ブログ記事内で外部URLをカード形式で表示するAstroコンポーネント。MDXファイルでインポートして使用する。「参考文献への感謝」を演出するホバーアニメーション付き。

---

## コンポーネント仕様

### ファイル
- **コンポーネント**: `src/components/LinkCard.astro`
- **使用場所**: MDXファイル（`.mdx`）内でインポートして使う

### Props

| Prop | 型 | 必須 | 説明 |
|------|----|------|------|
| `url` | `string` | ✅ | リンク先URL |
| `title` | `string` | ❌ | リンクのタイトル。省略するとビルド時にURLからfetchして`<title>`タグを取得 |

ドメイン名はURLから自動抽出して表示する（`new URL(url).hostname`）。

### タイトル自動取得の挙動

- `title` が指定されていない場合、ビルド時に `fetch(url)` でHTMLを取得し `<title>` タグの内容を使用する
- fetch失敗時（ネットワークエラー・タイムアウト等）はURLをそのままフォールバック表示する
- fetchはビルド時のみ（SSG）。実行時には発生しない

### 表示形式（V3ベース）

```
┌─────────────────────────────────────────────┐  ← 上部に左→右グラデーションライン
│  DOCS · BLENDER.ORG                         │  ← ドメイン（スモールキャップ）
│  オブジェクト情報 — Blender Manual    ↗    │  ← タイトル + 外部リンクアイコン
└─────────────────────────────────────────────┘
```

## ホバーアニメーション

ホバー時に以下が同時発動する：

1. **キラキラ（✦⋆）が舞い上がる**
   - アクセントカラー（#c4ad6c）の記号がカード上辺から上方へ消えるアニメーション
   - 5個、それぞれ微妙に delay をずらす

2. **「参考にさせていただきます 🙏」バッジが出る**
   - カード右上にアクセントカラーの吹き出しバッジがフェードイン
   - 下向きの三角吹き出しつき

3. **タイトルがアクセントカラーに変わる**

4. **ボーダーがアクセントカラーに（薄く）変わる**

## MDXへの移行

`src/pages/posts/2026-03-27-shading/index.md` を `.mdx` にリネームし、LinkCard を使用する。

### 使用例

```mdx
import LinkCard from '../../../components/LinkCard.astro'

<LinkCard
  url="https://docs.blender.org/manual/ja/dev/render/shader_nodes/input/object_info.html"
  title="オブジェクト情報 — Blender Manual"
/>
```
