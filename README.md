# ブログ運用方針

## 記事の粒度
- **1要求1記事**を基本とする
- 「〇〇したい」という単位で切る
    - ↑のような要求を満たすために必要になった周辺知識も記事内のキーワードとして追加する
- 1日の学習まとめはしない

## テンプレートの使い分け
- 自分向けのログ → **cornell**（雑でいい）
- 整理して共有したいとき → **article**

## ログモードのルール
- タイトルは考えない。やりたかったことをそのまま書く
- 画像・図は任意。なくていい
- 書くコストよりinputに全振りする

運用方針は、運用にモヤモヤを感じたら都度AIに相談しつつ柔軟に変えていく
